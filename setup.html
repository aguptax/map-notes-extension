2

Create a Google Cloud Project

  1. Go to Google Cloud Console — New Project
  2. Set project name to MapNotes
  3. Click Create and wait for it to finish
  4. Make sure the new project is selected in the top bar
3

Enable the Google Drive API

  1. Go to Google Drive API page
  2. Make sure your MapNotes project is selected in the top bar
  3. Click Enable
4

Set Up OAuth Consent Screen

  1. Go to OAuth Consent Screen
  2. Select External then click Create
  3. Fill in:
    • App name: MapNotes
    • User support email: your email
    • Developer contact: your email
  4. Click Save and Continue through the rest (Scopes, Test Users)
Note: While in "Testing" mode, only test users you add can sign in. Add your own Google email under Test Users.
5

Create OAuth Client ID

  1. Go to Credentials page
  2. Click + Create Credentials then OAuth Client ID
  3. Application type: Web application
  4. Name: MapNotes

  6. Click Create
Copy the Client ID that appears (it looks like 123456789-abcde.apps.googleusercontent.com). You need it for the next step.
6

Add Client ID to MapNotes
